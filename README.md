# 🚀 Personal Data Collector - Fundamental Booster

## 📖 Overview
Personal Data Collector is a beginner-friendly Python application that interacts with users, collects their personal details, performs basic calculations, and displays the information in a well-formatted way.

This project is designed to strengthen the fundamentals of Python programming by using real-world examples.

---

## 🎯 Project Goals

- Learn how to use 'input()' and 'print()'
- Work with different data types
- Store values using variables
- Perform arithmetic operations
- Apply type casting
- Use built-in functions like 'type()' and 'id()'
- Create a user-friendly interactive program

---

## ✨ Features

✅ Collects user details:
- Full Name
- Age
- Height
- Favourite Number

✅ Performs automatic calculations:
- Calculates approximate Birth Year
- Finds square of favourite number
- Calculates age after 5 years

✅ Displays:
- Data Type of each variable
- Memory Address using 'id()'
- Well-formatted summary
