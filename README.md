# 👤 Personal Data Collector

A beginner-friendly Python project that collects basic personal information from the user and demonstrates the use of Python data types, variables, user input, memory addresses, and simple calculations.

---

## 📖 Project Description

**Personal Data Collector** is a simple console-based Python application developed for beginners. It allows users to enter their personal details and then displays the collected information along with each value's data type and memory address. The program also estimates the user's birth year based on the entered age.

This project helps students understand the fundamentals of Python programming through practical implementation.

---

## 📖 About the Project

**Personal Data Collector** is a beginner-friendly Python console application that demonstrates the fundamentals of Python programming.

The application collects basic user information such as **Name**, **Age**, **Height**, and **Favourite Number**, then displays the entered information along with its **data type** and **memory address**. It also estimates the user's **birth year** based on the entered age.

This project was created to strengthen core Python concepts through practical implementation.
